# Finance Telegram Bot

Telegram-бот для управления личными финансами.

## Возможности

- Добавление доходов
- Добавление расходов
- Финансовая аналитика
- SQLite хранение данных
- Финансовые цели

## Стек технологий

- Python 3
- Aiogram 3
- SQLite
- Aiosqlite
- Matplotlib

## Запуск

```bash
pip install -r requirements.txt
py bot.py
